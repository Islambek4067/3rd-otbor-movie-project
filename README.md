## 🔑 Краткое руководство по API

### Base URL
```
https://api.themoviedb.org/3
```

### Использование API Key
К каждому запросу нужно добавлять `?api_key=YOUR_KEY`:
```
https://api.themoviedb.org/3/trending/movie/day?api_key=YOUR_KEY
```

### Изображения
```
Poster:   https://image.tmdb.org/t/p/w500/{poster_path}
Backdrop: https://image.tmdb.org/t/p/w1280/{backdrop_path}
Profile:  https://image.tmdb.org/t/p/w185/{profile_path}
```

### Основные эндпоинты

| Эндпоинт | Описание |
|---|---|
| `GET /trending/movie/day` | Трендовые фильмы за день |
| `GET /trending/tv/day` | Трендовые сериалы за день |
| `GET /search/multi?query=batman` | Поиск (фильм + сериал + человек) |
| `GET /movie/popular` | Популярные фильмы |
| `GET /movie/{id}` | Полная информация о фильме |
| `GET /movie/{id}/credits` | Список актеров |
| `GET /movie/{id}/videos` | Трейлеры (YouTube) |
| `GET /genre/movie/list` | Список жанров |
| `GET /discover/movie?with_genres=28` | Фильтр по жанрам |

> 📖 Полная документация API в файле: `api-docs.md`

## 📋 Задания

### Task 1 — Hero Section + Trending (8 баллов)
- Получение данных из API `/trending/movie/day`.
- Hero banner: фон (backdrop), заголовок, описание, рейтинг.
- Кнопка "Watch Trailer".
- Carousel: автоматическая смена каждые 5 секунд.

### Task 2 — Search (10 баллов)
- Поле поиска + debounce (300ms).
- Поиск через API `/search/multi`.
- Результаты: постер, название, год, рейтинг, тип (фильм/сериал).
- Индикатор загрузки + сообщение "Ничего не найдено".

### Task 3 — Movie Grid (8 баллов)
- Карточки популярных фильмов из `/movie/popular`.
- Адаптивная сетка (Grid): 5 / 3 / 2 колонки.
- Карточка: постер, название, рейтинг ⭐, год.
- Ховер-эффект: увеличение или оверлей.

### Task 4 — Фильтр по жанрам (8 баллов)
- Получение жанров из `/genre/movie/list`.
- Горизонтальная прокрутка чипов (тегов).
- При клике на жанр → фильтрация через `/discover/movie?with_genres=ID`.
- Стиль активного жанра.

### Task 5 — Модальное окно (12 баллов)
- Информация из `/movie/{id}`: хронометраж, бюджет, жанры, слоган.
- Список актеров (карусель: фото + имя + роль).
- Встроенный YouTube трейлер (iframe).
- Фон (backdrop) с блюром.
- Закрытие: кнопка + клавиша ESC.

### Task 6 — Пагинация (8 баллов)
- Кнопка "Load More" или бесконечный скролл.
- Параметр API `page`.
- Скелетон (Skeleton loader) при загрузке.

### Task 7 — Избранное (Favorites) (10 баллов)
- Кнопка ❤️ → сохранение в localStorage.
- Отображение секции избранного.
- Возможность удаления из списка.

### Task 8 — Табы + Сортировка (6 баллов)
- Переключение вкладок: Фильмы / Сериалы / Тренды.
- Сортировка: Популярность, Рейтинг, Дата выхода.

### Task 10 — Страница актера (10 баллов)
- Информация из `/person/{id}`: биография, дата рождения.
- Список работ (фильмография) с постерами.

### Task 9, 11, 12 — Адаптив, Темы и Анимации (20 баллов)
- Полный адаптив (Mobile, Tablet, Desktop).
- Переключение Тем (Dark/Light) через CSS переменные.
- Плавные анимации и переходы.


**Желаем удачи! 🚀**
